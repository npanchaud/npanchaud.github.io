Color Space Models

This page covers the basics about different color models, their strengths and their weakness in the context of web mapping and color transformations.

Why does it matter? These color space models try to mathematically represent color, but how we (human) perceive colors and their difference is tricky and far from simple.

RGB Red Green Blue

RGB stands for the red, green and blue light of this additive color model. It does not define what is meant with red, green, and blue colorimetrically, but how much of each color is included in the mix. Thus, RGB is a device-dependent color model and a RGB value does not define the same color across devices without some kind of color management.

Colors are expressed as RGB triplets (r,g,b), each component of which can vary from zero to a defined maximum value. If all the components are at zero the result is black; if all are at maximum, the result is the brightest representable white. The ranges can be expressed from 0 to 1 with any fractional values in between, or as percentage from 0% to 100%; or in computer as they are often stored as integer numbers from 0 to 255, which is the range of a single 8-bit byte.

HEX Hexadecimal notation

A hex triplet is a six-digit, three-byte hexadecimal number. The bytes represent the red, green and blue components of the color. One byte represents a number in the range 00 to FF (in hexadecimal notation). Web colors specify colors in the Truecolor (24-bit RGB) color scheme. It is often used to represent the RGB color model.
